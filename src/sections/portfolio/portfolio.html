Chow

ionic, google maps, angular, typescript, html, css

This is a personal project I took on to improve my knowledge in Angular and mobile design.

A conversation with a friend:

  • "You hungry?"
  • "Yes, what should we eat"
  • "I don't know"
  • "Me too"
The idea was simple, build something that would help people decide on what to eat.

Chow chooses a random eatery nearby the user's current location and if the user doesn't like the chosen option, they can ask for another option.

This idea enables users to explore restaurants they wouldn't normally try, which in turn, generates more traffic to more unknown local restaurants. I believe this app can help to enrich local communities.

Check out the app and lemme know what you think! (it looks better on mobile)

Time Map

google maps, fullcalendar, bootstrap, php, javascript, html, css

This was for my final year project where I created a timetable / map web app for my university campus.

The idea was to have students upload their lecture timetable into the app and it will populate the map with where their next classes were.

There was potential for this idea; where it could be used for things outside of the university campus. Where users could connect it to their own personal calendars (granted their events had locations) and it would be populated onto the map based on the times they had to be there.
I was given the opportunity to explore a little bit more of this idea during my summer internship with my university.

What I learned from this project:

  • Storing things in JSON is a thing
  • How to decipher documentation
  • How to work with APIs
